# Rest Timer

**1\. Rest Time**  
How long to rest between each set and exercise.  
Default: **1min30sec**

**2\. Early Reminder**  
How long before the rest timer is up to remind you about it.  
Default: **10sec**

**3\. Prep Time**  
How long before a timer starts (for Time based exercises).  
Default: **3sec**

**4\. Play Sound**  
If a sound should be played at the end of the rest timer.  
Default: **YES**

**5\. Enable Between Exercises**  
If the rest timer should be triggered between exercises.  
Default: **NO**

# Units

**1\. Weight**  
Select the unit system for recording weight.  
Default: **Imperial (lbs)**

**2\. Distance**  
Select the unit system for recording distance.  
Default: **Imperial (miles)**

# Keyboard

!!! note
    Requires PRO version

**1\. Use PRO keyboard**  
Indicates if the PRO keyboard should be used for data input.  
Default: **YES**

**2\. Keypad on Left**  
Indicates if the PRO keyboard should be left or right oriented.  
Default: **YES**

# Workout Data Entry

**1\. Default Sets**  
The default number of sets to add when adding a new exercise to a workout.  
Default: **5**

**2\. Default Reps**  
The default number of reps to set a set when adding a new exercise to a workout.  
Default: **7**

**3\. Default Time**  
The default time to set time based exercises to when adding a new exercise to a workout.  
Default: **5min**

**4\. Continue With Last Workout Values**  
Indicates if a new workout should continue with values from a similary previous workout.  
Default: **YES**

# Workout Data Export 

**1\. Restore Default Exercises**  
Restores the default exercies. Your created exercises will not be affected.

**2\. Restore Default Workouts**  
Restores the default workouts. Your completed and created workouts will not be affected.

**3\. Export Workouts to CSV**  
Exports all your completed workouts to a CSV file format. 

**4\. Export Exercises to CSV**  
Export your entire Exercise catalog to a CSV file format. 

**5\. Export Data**  
Export your entire data to a FitNotes readable file format.

**6\. Export as CSV To E-Mail**  
Upon each workout completion exports that workouts data as CSV and mails it to the provided list.

!!! note
    This feature requires PRO version
